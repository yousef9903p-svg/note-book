A Notebook Web App that lets users type notes, save them to the browser’s local storage, reload them later, and switch between light/dark mode.

⚙️ Features
Text Input Area: Users can write notes in a styled box.

Save Button: Stores the note in localStorage and shows a “save successful” message.

Load Button: Retrieves the saved note back into the input box.

Dark Mode Toggle: A “night vision” button switches the site’s theme to dark mode.

Animations: The save message fades in/out with a smooth animation.

🛠️ Technologies Used
HTML/CSS: For layout and styling (rounded input box, buttons, dark mode theme).

JavaScript: For event handling, saving/loading notes, and toggling classes.

LocalStorage: To persist notes across sessions.

🚀 How It Works
User writes a note in the input box.

Clicking Save stores the note in local storage and clears the box.

Clicking Load retrieves the saved note back into the box.

Clicking Night Vision toggles dark mode styling.

This is a great beginner project because it teaches DOM manipulation, local storage, and styling with CSS animations.
